5动画

默认情况下，所有的 HTML 元素有一个静态的位置，且是不可移动的。 如果需要改变为，我们需要将元素的 position 属性设置为 relative, fixed, 或 absolute!

点击 "开始" 按钮开始动画。

点击 "停止" 按钮停止当前激活的动画，但之后我们能再动画队列中再次激活。

点击 "停止所有" 按钮停止当前动画，并清除动画队列，所以元素的所有动画都会停止。

点击 "停止动画，但完成动作" 快速完成动作，并停止它。\n

3 jqueryDOM

1DOM 遍历

    parent() 被选元素的直接父元素。
    parents() 返回被选元素的所有祖先元素，它一路向上直到文档的根元素 ()。
    parentsUntil() 返回介于两个给定元素之间的所有祖先元素。
    children() 返回被选元素的所有直接子元素。
    find() 返回被选元素的后代元素，一路向下直到最后一个后代。

    同胞:
    siblings() 返回被选元素的所有同胞元素。使用可选参数来过滤对同胞元素的搜索。
    next() 被选元素的下一个同胞元素。
    nextAll()  返回被选元素的所有跟随的同胞元素。可用输入参数过滤。
    nextUntil() 返回介于两个给定参数之间的所有跟随的同胞元素。
    prev()   和next类似。只不过是上一个，向前。方向相反
    prevAll()
    prevUntil()

    过滤： 
    first() 返回被选元素的首个元素。
    last() 方法返回被选元素的最后一个元素。  
    filter() 方法允许您规定一个标准。不匹配这个标准的元素会被从集合中删除，匹配的元素会被返回。
    not() 方法返回不匹配标准的所有元素。not() 方法与 filter() 相反。

    如： $("h2").siblings().first().css({"color":"red","border":"2px solid red"}); //h2所有同胞的第一个
        $("div p").last().css("background-color","yellow"); //最后一个div p元素
        $("p").filter(".url");//带有类名 "url" 的所有 p 元素


  里面的可选参数可进行筛选。
  如$("span").parents("ul");  选出ul 
  $("div").children("p.1");  选出 p  且class为1
  $("div").find("*");  所有后代
  $("div").find("span"); 选出span
  $("div").find(); 什么都选不出来

  语法:
$(selector).load(URL,data,callback);
  必需的 URL 参数规定您希望加载的 URL。
  可选的 data 参数规定与请求一同发送的查询字符串键/值对集合。
  可选的 callback 参数是 load() 方法完成后所执行的函数名称。
